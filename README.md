# Customer Behavior Data Analyst Portfolio Project 📊
This project demonstrates a full data analytics lifecycle similar to real industry practice. It covers key stages of analysis, including data preparation, transformation, exploration, visualization, and reporting to support data-driven decision-making.

## Overview

This project demonstrates an end-to-end data analytics workflow, including data loading, exploratory data analysis (EDA), data cleaning, SQL querying, dashboard visualization, and reporting. The goal is to extract meaningful business insights and present them in an easy-to-understand format for decision-making.

## Dataset

- The dataset contains transactional/customer behavior data.
- It was loaded into Python for preprocessing and analysis.

## Tools Used 🛠️
* Python (Pandas, NumPy, Matplotlib/Seaborn)
- SQL (PostgreSQL / MySQL / SQL Server)
- Power BI
- Gamma (for presentation creation)

## Project workflow

1. Loaded dataset into Python.
2. Performed exploratory data analysis (EDA).
3. Cleaned data by handling missing values and inconsistencies.
4. Performed feature engineering where necessary.
5. Stored and queried data using SQL.
6. Built interactive dashboards in Power BI.
7. Generated business report and presentation slides.

# Dashboard Highlights 📈
The interactive dashboard provides insights into:
- Revenue and spending patterns
- Customer segmentation
- Product performance
- Subscription behavior
- Gender-based or category-based analysis

# Results

- Identified key customer and product trends.
- Analyzed revenue contributions across segments.
- Evaluated impact of discounts and subscriptions.
- Provided data-driven business insights.

# How to Run the Project

1. Clone the repository.
2. Install required Python libraries using:
```pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2```
3. Open the Jupyter Notebook file.
4. Run cells sequentially to reproduce the analysis.
5. Import cleaned data into your SQL database if needed.
6. Open the Power BI file to view the dashboard.
